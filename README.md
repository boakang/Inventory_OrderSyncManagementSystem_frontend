# Inventory Order Sync Management System - Frontend

## Project Structure

The project is organized as follows:

```
src/
  App.js                # Main application entry point
  pages/
    CustomersPage.js    # Page for managing customers
    InventoryPage.js    # Page for managing inventory
    OrdersPage.js       # Page for managing orders
    ProductsPage.js     # Page for managing products
    ReportsPage.js      # Page for viewing reports
```

## Description

This is the frontend for the Inventory Order Sync Management System. It is structured to separate concerns by organizing different pages into the `pages/` directory. The `App.js` file serves as the main entry point for the application.

### Key Files:
- **App.js**: The main entry point of the application.
- **pages/**: Contains all the individual pages of the application.
  - **CustomersPage.js**: Handles customer-related functionality.
  - **InventoryPage.js**: Handles inventory-related functionality.
  - **OrdersPage.js**: Handles order-related functionality.
  - **ProductsPage.js**: Handles product-related functionality.
  - **ReportsPage.js**: Handles report-related functionality.

## How to Use

1. Clone the repository.
2. Navigate to the project directory.
3. Install dependencies (if applicable).
4. Start the development server to view the application in the browser.

## Notes

- Ensure that the backend for the Inventory Order Sync Management System is running to fully utilize the features of this frontend application.
